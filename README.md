# Weight & Balance Calculator (C172)

Python program that calculates aircraft weight and balance, centre of gravity at takeoff and landing, and generates a flight briefing report for a Cessna 172.

## Features
- Takes pilot, passenger, and baggage weight as input
- Calculates centre of gravity at different flight stages
- Checks aircraft weight and balance limits
- Generates a flight briefing report file

## How to Run
```bash
python weight_balance.py
